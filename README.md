# 🍎 GoldenAppleCooldown

Un plugin simple y ligero para **PocketMine-MP (API 2.0.0)** que añade un tiempo de espera al comer manzanas doradas, configurable por mundo. Perfecto para evitar abusos en Survival o Factions sin complicar tu servidor.

---

### ⚡ ¿Qué hace?
*   Bloquea el consumo de **Manzana Dorada** y **Manzana Dorada Encantada** si el jugador está en cooldown.
*   Muestra un mensaje flotante (popup) con los segundos restantes.
*   Funciona **solo en los mundos** que tú elijas.
*   Notifica al jugador cuando puede volver a comer.
